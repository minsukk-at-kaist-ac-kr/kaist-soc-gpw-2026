# KAIST SoC GPW 2026 Website

Static website files for KAIST SoC Global Preview Week 2026.

Upload the contents of this package to the root of the GitHub Pages repository. The repository should contain:

- `index.html`: minimal invitation-based landing page.
- `assets/`: shared images and logo.
- `gpw-*/index.html`: school-specific invitation/application pages.
- `robots.txt`: prevents search indexing.
- `CNAME`: GitHub Pages custom domain configuration.

GA4 tag installed with Measurement ID `G-GB5215L1R8` on all HTML pages. Use GA4 Pages and screens → Page path and screen class to distinguish school-specific pages.
